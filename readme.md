# nail-ai-ops

美甲 AI 试戴与智能运营助手  
Meituan AI Hackathon Project

## 1. 项目简介

`nail-ai-ops` 是一个面向单店美甲商家的 AI 智能运营与客户端试戴展示系统。

在美甲到店服务场景中，用户经常会在平台上浏览大量美甲款式图，但难以判断款式是否适合自己的手型、肤色和个人风格，导致决策成本高、咨询转化慢、预约意愿不稳定。另一方面，商家虽然拥有大量款式图和历史经验，但在爆款识别、款式主推、营销文案生成和上新策略制定上仍然高度依赖人工判断，运营效率有限。

本项目希望通过 AI 能力连接用户侧和商家侧：  
用户可以通过自然语言描述需求、筛选风格、选择款式，并上传手图或使用示例手图生成美甲试戴效果；商家可以基于店铺款式库和运营目标，由 AI 生成主推策略、上新建议和宣传文案，从而提高款式展示效率和用户转化效率。

## 2. 项目定位

本项目不做大而全的本地生活平台，而是聚焦 Hackathon 最小可行产品：

> 单店美甲 AI 智能运营助手 + 客户端试戴展示页

核心目标是完成一个可以演示、可以解释、可以迭代的业务闭环：

```text
商家款式库
→ AI 理解款式风格
→ 生成运营策略和展示内容
→ 客户端展示推荐款式
→ 用户上传手图 / 使用示例手图
→ 生成美甲试戴结果
→ 用户咨询 / 预约 / 换一款
