## 基础匹配规则

### 1. 肤色匹配规则

#### 1.1 冷色调肤色（粉调）
**特征：** 手腕血管呈蓝色或紫色，银色首饰更衬肤色

**推荐颜色：**
- **首选：** 冷色系（蓝色、紫色、粉色）
- **次选：** 中性色（灰色、银色）
- **慎用：** 暖橙色、金黄色

**推荐款式：**
- 韩式渐变（粉色系）
- 猫眼效果（冷色调）
- 法式美甲（经典白边）

#### 1.2 暖色调肤色（黄调）
**特征：** 手腕血管呈绿色，金色首饰更衬肤色

**推荐颜色：**
- **首选：** 暖色系（红色、橙色、金色）
- **次选：** 大地色系（棕色、裸色）
- **慎用：** 冷蓝色、冷紫色

**推荐款式：**
- 经典红色系
- 金属质感款
- 大地色系渐变

#### 1.3 中性肤色
**特征：** 手腕血管呈蓝绿色，金银首饰都适合

**推荐颜色：**
- **百搭：** 几乎所有颜色
- **推荐：** 裸色系、粉色系
- **可尝试：** 各种流行色

**推荐款式：**
- 经典法式
- 极简风格
- 流行艺术款

### 2. 手型适配规则

#### 2.1 手指修长型
**特征：** 手指纤细修长，骨节不明显

**推荐款式：**
- **长度：** 中长款、长款、超长款
- **风格：** 欧美风、艺术创意
- **工艺：** 立体雕花、复杂设计
- **避免：** 过短的指甲设计

**具体推荐：**
- 欧美艺术款（长款）
- 立体雕花款
- 复杂图案设计

#### 2.2 手指娇小型
**特征：** 手指短而小巧，掌心较小

**推荐款式：**
- **长度：** 短款、中长款
- **风格：** 日韩系、可爱风
- **工艺：** 简约设计、小巧装饰
- **避免：** 过于夸张的设计

**具体推荐：**
- 韩式渐变款
- 法式美甲
- 简约线条款

#### 2.3 手指偏粗型
**特征：** 手指较粗，骨节明显

**推荐款式：**
- **长度：** 中长款（避免过短）
- **风格：** 经典款、极简风
- **工艺：** 竖向线条、渐变色
- **避免：** 横向线条、复杂装饰

**具体推荐：**
- 法式美甲（视觉拉长）
- 竖向渐变款
- 简约纯色款

#### 2.4 手掌宽大
**特征：** 手掌较宽，手指相对较短

**推荐款式：**
- **长度：** 中长款
- **风格：** 简约大气
- **工艺：** 简单设计，避免过于精致
- **避免：** 过于小巧的装饰

**具体推荐：**
- 经典纯色款
- 简约法式
- 大气几何款

## 特殊情况处理

### 1. 新用户推荐
**策略：** 基于基础信息进行保守推荐

**推荐流程：**
1. 收集基础信息（肤色、手型、年龄）
2. 选择经典安全款式
3. 提供多个选项供选择
4. 根据反馈调整后续推荐

**推荐款式：**
- 经典法式美甲
- 裸色纯色款
- 浅粉色渐变款

### 2. 无明确偏好用户
**策略：** 基于场景和基础匹配推荐

**推荐流程：**
1. 确定主要使用场景
2. 根据基础匹配筛选
3. 提供3-5个不同风格选项
4. 引导用户选择偏好

### 3. 特殊需求处理

#### 过敏体质
**处理策略：**
- 推荐低敏材质
- 避免复杂化学工艺
- 选择天然成分产品

#### 经常洗手
**处理策略：**
- 推荐持久性强的款式
- 避免容易脱落的装饰
- 选择耐磨性好的质地

#### 指甲脆弱
**处理策略：**
- 推荐短款设计
- 避免过长指甲
- 选择加固型产品
