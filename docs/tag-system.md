# 美甲款式标签体系

## 概述

本标签体系为美甲AI试戴与推荐系统提供标准化的分类标准，涵盖美甲款式的核心维度，支持精准筛选与智能推荐功能。

## 标签维度

### 1. 风格分类 (style)
美甲的整体设计风格，反映美学特征和文化元素。

**枚举值：**
- **经典款** - 传统简约设计，适合日常佩戴
- **法式美甲** - 经典法式边设计，优雅百搭
- **日式美甲** - 日式可爱风，注重细节和立体感
- **韩式美甲** - 韩系清新风格，色彩柔和
- **欧美风** - 大胆前卫，色彩丰富
- **复古风** - 怀旧复古元素，怀旧色调
- **极简风** - 简约线条，少即是多
- **艺术风** - 创意艺术表现，独特设计
- **自然风** - 自然元素，清新淡雅
- **奢华风** - 高端奢华，钻饰丰富
- **甜美风** - 可爱甜美，粉色系为主
- **酷飒风** - 个性酷炫，深色系
- **民族风** - 民族图案，文化元素
- **几何风** - 几何图形，现代感强

### 2. 颜色分类 (color)
美甲的主色调和配色方案，使用专业色名。

**主色系：**
- **透明色** - 透明或半透明底色
- **裸色** - 自然肤色系，含裸粉、裸棕、裸灰
- **粉色系** - 粉红、玫瑰粉、豆沙粉、珊瑚粉
- **红色系** - 正红、酒红、樱桃红、砖红
- **橙色系** - 橙色、珊瑚橙、南瓜橙
- **黄色系** - 明黄、柠檬黄、香槟金
- **绿色系** - 薄荷绿、墨绿、橄榄绿、翠绿
- **蓝色系** - 天蓝、宝蓝、深蓝、雾霾蓝
- **紫色系** - 薰衣草紫、葡萄紫、深紫
- **棕色系** - 咖啡色、巧克力色、焦糖色
- **灰色系** - 浅灰、深灰、银灰
- **黑色** - 纯黑色
- **白色** - 纯白色
- **金银色** - 金色、银色

**配色方案：**
- **单色** - 单一颜色
- **双色** - 两种颜色搭配
- **多色** - 三种及以上颜色
- **渐变色** - 颜色渐变效果
- **撞色** - 对比色搭配
- **同色系** - 同一色系深浅搭配

### 3. 长度分类 (length)
指甲的长度分类，影响整体视觉效果。

**枚举值：**
- **短款** - 指甲长度适中，甲床露出较多
- **中长款** - 指甲稍长，甲床露出适中
- **长款** - 指甲较长，甲床露出较少
- **超长款** - 指甲很长，艺术性强

### 4. 质地分类 (texture)
美甲表面的质感和特殊效果。

**基础质地：**
- **亮面** - 光泽感强，反光效果好
- **哑光** - 无光泽，质感柔和
- **半哑光** - 介于亮面和哑光之间

**特殊效果：**
- **猫眼** - 猫眼石效果，光线变化
- **闪粉** - 含有闪粉颗粒，闪烁效果
- **珠光** - 珍珠光泽，温润质感
- **金属** - 金属质感，银色/金色
- **磨砂** - 磨砂质感，细腻纹理
- **果冻** - 果冻质感，Q弹透明
- **镜面** - 镜面效果，高度反光

### 5. 工艺分类 (craft)
美甲的制作工艺和技术特点。

**基础工艺：**
- **纯色** - 单一颜色涂抹
- **渐变** - 颜色渐变过渡
- **法式** - 法式边设计
- **晕染** - 颜色晕染效果
- **大理石** - 大理石纹路
- **水波纹** - 水波纹理

**装饰工艺：**
- **贴纸** - 美甲贴纸装饰
- **手绘** - 手工绘画图案
- **雕花** - 立体雕花工艺
- **贴钻** - 水钻装饰
- **铆钉** - 铆钉装饰
- **链条** - 链条装饰
- **珍珠** - 珍珠装饰
- **立体装饰** - 立体造型装饰

**特殊工艺：**
- **光疗** - 光疗延长
- **水晶** - 水晶延长
- **贴片** - 甲片装饰
- **3D** - 3D立体效果
- **镂空** - 镂空设计

### 6. 场景分类 (scene)
美甲的适用场景和场合。

**日常场景：**
- **日常** - 日常生活佩戴
- **通勤** - 职场办公场合
- **约会** - 约会聚会
- **休闲** - 休闲娱乐

**特殊场合：**
- **婚礼** - 新娘美甲，婚礼场合
- **派对** - 派对聚会，夜生活
- **节日** - 节日庆典
- **度假** - 度假旅游
- **晚宴** - 正式晚宴
- **舞台** - 舞台表演

**季节场景：**
- **春季** - 春季限定款式
- **夏季** - 夏季清爽款式
- **秋季** - 秋季温暖款式
- **冬季** - 冬季厚重款式

### 7. 人群分类 (crowd)
美甲款式适合的典型用户群体，用于推荐解释和卡片文案。

**枚举值：**
- **学生党** - 偏好日常、上课、休闲和低负担款式
- **上班族** - 偏好通勤、简约、耐看和正式场景款式
- **新娘** - 偏好婚礼、拍照和仪式感款式
- **时尚人群** - 偏好派对、舞台和高辨识度款式
- **个性玩家** - 偏好艺术风、酷飒风和强表达款式
- **大众用户** - 适合广泛人群的百搭款式

## 标签使用规则

### 1. 必选标签
每款美甲必须包含以下标签：
- 1个风格标签
- 1个颜色标签
- 1个工艺标签
- 1-3个场景标签
- 1-3个适合人群标签

### 2. 可选标签
根据美甲特点可选择添加：
- 1个长度标签
- 1个质地标签

### 3. 标签优先级
当一款美甲符合多个标签时，按以下优先级选择：
1. **风格** - 选择最突出的主要风格
2. **颜色** - 选择主色调，次要色调在描述中体现
3. **工艺** - 选择最复杂的工艺为主标签
4. **场景** - 选择最典型的主要场景

### 4. 命名规范
- 标签名称使用中文，简洁明了
- 避免使用主观性过强的词汇
- 同一标签在不同维度下不重复
- 标签名称保持一致性，不使用同义词
